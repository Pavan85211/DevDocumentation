## **1. What is an AWS CloudWatch Dashboard?**  
AWS **CloudWatch Dashboards** provide a **customizable, real-time visual representation** of AWS resources, metrics, and logs. You can monitor performance, detect issues, and analyze trends across multiple AWS services in a **single view**.  

### **Why Use CloudWatch Dashboards?**  
‚úÖ **Centralized monitoring** ‚Äì View multiple AWS services in one place  
‚úÖ **Real-time insights** ‚Äì Track key performance indicators (KPIs)  
‚úÖ **Custom visualizations** ‚Äì Graphs, metrics, logs, and alarms  
‚úÖ **Multi-region support** ‚Äì Monitor AWS resources across regions  
‚úÖ **Share with teams** ‚Äì View dashboards across AWS accounts

## **2. Key Features of CloudWatch Dashboards**  
### ‚úÖ **Custom Widgets**
- **Metric Widgets** ‚Äì Line charts, bar charts, number gauges  
- **Text Widgets** ‚Äì Annotations, custom notes  
- **Log Insights Widgets** ‚Äì Display logs from **CloudWatch Logs Insights**  
- **Alarm Status Widgets** ‚Äì Show active alarms  

### ‚úÖ **Multi-Region & Multi-Account Monitoring**
- Add widgets from **multiple AWS regions**  
- Monitor **multiple AWS accounts** using AWS Organizations  

### ‚úÖ **Integration with AWS Services**
- **EC2** ‚Äì CPU, memory, network usage  
- **RDS** ‚Äì Database connections, read/write latency  
- **Lambda** ‚Äì Invocation count, errors, duration  
- **ECS/EKS** ‚Äì Container performance metrics  
- **S3** ‚Äì Storage usage, requests per second

## **6. Automating Dashboards with AWS CLI & Terraform**  

### **Create a Dashboard using AWS CLI**
```sh
aws cloudwatch put-dashboard --dashboard-name "EC2-Monitoring" --dashboard-body '{
    "widgets": [
        {
            "type": "metric",
            "properties": {
                "metrics": [
                    ["AWS/EC2", "CPUUtilization", "InstanceId", "i-1234567890abcdef0"]
                ],
                "title": "EC2 CPU Utilization"
            }
        }
    ]
}'
```

### **Create a Dashboard using Terraform**
```hcl
resource "aws_cloudwatch_dashboard" "example" {
  dashboard_name = "EC2-Dashboard"

  dashboard_body = jsonencode({
    widgets = [
      {
        type = "metric"
        properties = {
          metrics = [
            ["AWS/EC2", "CPUUtilization", "InstanceId", "i-1234567890abcdef0"]
          ]
          title = "EC2 CPU Utilization"
        }
      }
    ]
  })
}
```
